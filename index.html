Project Moonstone

Uncover hidden relationshps in health data

General summary of tool: With access to a health population database, a user has the option to select two data points and plot their relationship. I.e. Option 1: age, Option 2: Height. The program will acess data from a server and plot the results.

Project Workflow

  1. Write SQL to access a health database. Database must contain a ton of quantifiable quantifiable health metrics.
    1. Where can I obtain such a database?
  2. Display dropdown list of potential traits that the user can select
  3. Record users selection of two traits, upon them running the search
  4. Write SQL to gather and store the list
  5. Write Python to parse through the numbers
  6. Display the relationship in a graph, State the R^2 Value
