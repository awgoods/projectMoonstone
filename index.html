Project Moonstone

Uncover hidden relationshps in health data

The 21st buzzwords of IoT and big-data are used so often and interchangably the average person tends to forget they're actual things that can help society. While their applications span across any-and-all industries (both technical and humanitarian), one area that excites me especially is Mobile Health, wearable technology, and remote patient monitoring (See the Health Data wiki page). All three of these areas have one thing in common: they create patient data. In this demonstration, witness first-hand how large sets of health metrics can help researchers glean insights about health trends and relationships.

Here's the sitch: you're the reseacher. Pick two health metrics below and discover any potential relationships among them. The program will access HealthData.gov's Database, gather the data, and display it for you in a nice little plot. See if you can uncover any hidden relationships! (Hint: The higher the R2 value, the more related the two variables are!)

Project Workflow

  1. Write SQL (the most popular RDBMS) to access a health database.
  2. Display dropdown list of potential traits that the user can select
  3. Record users selection of two traits, upon them running the search
    1. You can mix-and-match databases, to provide more variety and freedom in your analysis
  4. Write SQL to gather and store the list
  5. Write Python (Django or Flask) to parse through the numbers
  6. Display the relationship in a graph, State the R^2 Value
